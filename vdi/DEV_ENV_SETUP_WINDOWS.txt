# Intellij Idea Installtion

https://www.jetbrains.com/idea/download/?section=windows



# Download and install maven 3.9.9 (Optional) - You may use embeded maven in intellij
1. Download apache-maven-3.9.9-bin.zip
https://maven.apache.org/download.cgi

2. Extract the downloaded file to a desired folder (e.g. C:\AI\maven)

3. Launch Intellij Idea anc click on settings icon (typically bottom left)
Go to Build, Exceution, Demployment > Build Tools > Maven
Select Maven home path to the Maven folder.

4. Apply the sttings and Exit Intellij Idea

5. Add maven bin path to the User PATH variable (e.g. C:\AI\maven\bin)

# Configure maven settings 
1. Create a settings.xml file under C:\Users\{user}\.m2\
Enter proxy information
Ref: https://maven.apache.org/guides/mini/guide-proxies.html

2. open "hmdm-server" folder as project in Intellij Idea

3. Go to intellij Settings
Go to Build, Exceution, Demployment > Build Tools > Maven > runner
a) JRE path should be path zulu8 path
b) choose skip tests

4. right click on project root > module settings > Project
a) Choose JDK 1.8, Language Level 8

5. Open hmdm-server\server and make a copy of build.properties.example in the same folder
   rename to build.properties (ignore if file is already present)
   
# Build:
1. open "hmdm-server" folder as project in Intellij Idea 
2. Go to Maven lifecycles and click on install
